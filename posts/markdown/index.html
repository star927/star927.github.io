如何在Markdown中画流程图

typora 画流程图、时序图(顺序图)、甘特图

Flowchart Orientation 

flowchart TB
top --> bottom
flowchart TD
top_down --> same_as_top_to_bottom
flowchart BT
bottom --> top
flowchart RL
right --> left
flowchart LR
left --> right

Node shapes 

flowchart TB
text <--> id1[Hello\nWorld!] --> id2(text) --> id3([Hello World!])
id4[[text]] --> id5[(Database)] --> id6((circle)) --> id7>text]
id8{text} --> id9{{text}}
id10[/text/] --> id11[\text\] --> id12[\text/] --> id13[/text\]

Links between nodes 

flowchart TD
A1-->B1
A2---B2
A3-- This is the text! ---B3
A4---|This is the text|B4
A5-->|text|B5
A6-- text -->B6
flowchart TD
A1-.->B1
A2-. text .-> B2
A3 ==> B3
A4 == text ==> B4
flowchart TD
A -- text --> B -- text2 --> C
a --> b & c--> d
m & n--> x & y
flowchart TD
A1 --o B1
A2 --x B2
A3 o--o B3
A4 <---> B4
A5 x----x B5
